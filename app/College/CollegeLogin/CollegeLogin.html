College Log In
Instructions सूचना मराठीमध्ये
  • PUNCODE: Use your college puncode for login.
  • Password: Use your BCUD login password here.
  • You can do below things from your login:
    1. Temporary Number Generation: You can generate temporary number for aadhaar for thoes students who is not enrolled for aadhaar.
    2. Email Id and Mobile Number Update: You can update email id and mobile number of students by using OTP verification.
    3. Course Dropout Request: You can send request for course dropout for thoes students who want to change their course.
सूचना Instructions in English
  • PUNCODE: Login साठी आपल्या कॉलेजचा PUNCODE वापरा.
  • Password: येथे आपला BCUD Login चा Password वापरा.
  • आपल्या Login मधून आपण खालील गोष्टी करू शकता:
    1. Temporary Number Generation: आधारसाठी नोंदणी नसलेल्या विद्यार्थ्यांसाठी आपण Temporary number तयार करू शकता.
    2. Email Id and Mobile Number Update: आपण OTP पडताळणी करून विद्यार्थ्यांचा Email Id आणि Mobile number अद्ययावत करू शकता.
    3. Course Dropout Request: जे विद्यार्थी आपला कोर्स बदलू इच्छितात त्यांच्यासाठी आपण कोर्स सोडण्याची विनंती पाठवू शकता.
